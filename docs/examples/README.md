# Examples (Skeleton)

이 디렉터리는 규칙 문서(`docs/rules/`)와 쌍으로 사용하는 예시 모음이다.
현재는 Controller/Service/Repository 3축의 뼈대만 만들고, 예시는 추후 라운드에서 채운다.

## 구조
- `docs/examples/controller/README.md`
- `docs/examples/service/README.md`
- `docs/examples/repository/README.md`

## 작성 원칙
- 예시는 최소 실행 단위로 작성한다.
- "좋은 예시"와 "피해야 할 예시"를 함께 둔다.
- 규칙 문서의 항목 번호/제목을 본문에 링크한다.
